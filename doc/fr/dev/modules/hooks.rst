La gestion des hooks
====================

Un hook permet d'enrichir l'interface utilisateur par des informations provenant d'autres modules.
Attention, il ne faut pas confondre un Hook avec un Event. Le Hook est visuel tandis qu'un Event
est un évènement émis à un instant précis dans l'application (par exemple, après un ajout de Host).
