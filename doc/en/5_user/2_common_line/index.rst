CentreonConsole (command line)
==============================

Centreon provide natively a console (command line) user interface in order to improve integration of Centreon in all information systems. CentreonConsole is easy to use in order to give to sysadmins a low level inteface but without having the feeling to develop.

CentreonConsole provides a list of commands to manage centreon objects.

The list of object commands is available with the following commandline :

::

   <centreon_path>/external/bin/centreonConsole -l
