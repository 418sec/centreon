From sources¶

In order to upgrade Centreon from sources, download the latest Centreon package.

Shell installation¶

Extract the package:

$ tar xvfz centreon-2.x.x.tar.gz

Change the directory:

$ cd centreon-2.x.x

Run the upgrade script:

$ ./install -u /etc/centreon

Where /etc/centreon is to be replaced by configuration directory.

Prerequisites check¶

If [Step 01] is successful, you should not have any problem here. Otherwise, go back to [Step 01] and install the prerequisites:
